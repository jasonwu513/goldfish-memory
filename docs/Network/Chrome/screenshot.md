# Chrome 長截圖

## 使用 Chrome 開發者工具


### Mac 

1. 快捷鍵: `Command + Option + I` or  MX Keys F4
2. Run Command: `Command + Shift + P`
3. 輸入 `Capture full size screenshot`

### Windows

1. 快捷鍵: `F12`
2. Run Command: `Ctrl + Shift + P`
3. 輸入 `Capture full size screenshot`

不用輸入 Capture full size screenshot 只要打 full 就會出現
